# RV32I-Simulator

## Projeto de Simulação da Arquitetura RISC-V (RV32I)

Este projeto consiste no desenvolvimento de um simulador de computador com a arquitetura de conjunto de instruções **RISC-V (ISA RV32I)**, conforme os requisitos da disciplina de Organização de Computadores da Universidade Católica de Santos- Unisantos/ Walter Oliveira.

# Objetivo

O objetivo é implementar os pricipais blocos de um computador de ciclo único, gerenciados por um barramento, e demonstrar o ciclo de busca e execução de instruções.
